Introduction to std::string in C++

The std::string class provides a modern, safe, and convenient way to work with text in C++. Unlike C-style strings (character arrays), std::string automatically manages memory, grows dynamically, and provides many useful built-in methods.

Since you have already used C-strings, think of std::string as a higher-level abstraction that removes many of the manual and error-prone tasks such as buffer management and null terminators.

Creating and Using std::string

To use std::string, include the <string> header. 

#include <iostream>
#include <string>
using namespace std;

int main() {
    string name = "Alice";

    cout << name << endl;

    return 0;
}
  • No need for manual memory allocation
  • No need for null terminators ('\0')
  • Works seamlessly with streams like cout
Length and Character Access

You can easily inspect and access characters in a string. 

string s = "hello";

cout << s.length() << endl;   // Outputs: 5
cout << s.size() << endl;     // Same as length()

cout << s[0] << endl;        // 'h'
cout << s.at(1) << endl;     // 'e'

Note: at() performs bounds checking, while [] does not.

Concatenation and Appending

Strings can be combined easily using operators or member functions. 

string first = "Hello";
string second = "World";

string result = first + " " + second;
cout << result << endl;  // Hello World

first += "!";
cout << first << endl;   // Hello!

This is much simpler than using strcat() with C-strings.

Commonly Used Methods
  • .length() / .size() — returns number of characters
  • .substr(pos, len) — returns a substring
  • .find(str) — finds position of substring
  • .append(str) — adds text to the end
  • .clear() — removes all characters
  • .empty() — checks if string is empty
string text = "C++ Programming";

cout << text.substr(0, 3) << endl;   // C++
cout << text.find("Pro") << endl;   // position of "Pro"

text.append(" is fun");
cout << text << endl;

text.clear();
cout << text.empty() << endl;     // 1 (true)
Comparing Strings

Unlike C-strings, you can directly compare std::string values using operators. 

string a = "apple";
string b = "banana";

if (a == b) {
    cout << "Equal";
}

if (a < b) {
    cout << "apple comes before banana";
}

This is much safer and simpler than using strcmp().

Working with C-Strings

You may need to interact with legacy code that uses C-style strings. 

string s = "hello";

const char* cstr = s.c_str();

cout << cstr << endl;

c_str() provides a read-only C-style string view of the data.
