Assignment: Palindrome Checker (std::string)

A palindrome is a word, phrase, or sequence that reads the same forward and backward after removing spaces, punctuation, and differences in letter case.

Examples:

  • racecar
  • madam
  • A man, a plan, a canal: Panama
  • No lemon, no melon

In this assignment, you will build a program using std::string to determine whether a given input is a palindrome under these rules.

Program Requirements
  • Prompt the user to enter a full line of text (use getline()).
  • Store the input using std::string.
  • Process the string so that:
    • All uppercase letters are treated as lowercase.
    • All spaces are removed.
    • All punctuation is ignored (only letters and/or digits should remain).
  • Create a reversed version of the processed string.
  • Compare the processed string with its reversed version.
  • Output whether the input is or is not a palindrome.
Functional Expectations
  • Use std::string methods such as:
    • .length() or .size()
    • Character access with [] or .at()
    • Concatenation or building new strings
  • Use a loop to examine each character in the string.
  • Use conditional logic to filter out unwanted characters.
  • Do NOT use external libraries beyond standard headers covered in class.
Example Program Behavior
Enter a phrase: A man, a plan, a canal: Panama
Result: This is a palindrome.

Enter a phrase: Hello World
Result: This is NOT a palindrome.
Coding Standards & Requirements
  • Use the course-standardized header comment block at the top of your file.
  • Include appropriate headers (<iostream>, <string>, etc.).
  • Use meaningful variable names (e.g., original, cleaned, reversed).
  • Indent and format your code consistently.
  • Include inline comments explaining key steps of your logic.
  • Test your program with multiple inputs, including edge cases.
Hints (Optional Guidance)
  • Use tolower() to normalize character case.
  • Check characters with functions like isalpha() or isdigit().
  • Build a cleaned version of the string before reversing it.
  • Consider using a loop to construct the reversed string manually.
