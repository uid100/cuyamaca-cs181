Lab #11 - std::string

Lab Introduction: Working with std::string in C++

In this lab, you will apply your knowledge of std::string to complete two exercises that involve creating and using string objects in C++.

Write functions and simple programs to practice and demonstrate your understanding of std::string, including creating and manipulating string objects.

 

General Requirements

      Put each exercise in its own directory.
      Each class should be in its own source file (.cpp) and header file (.h).
      Use appropriate access specifiers (public, protected, private) for class members.
      Don't forget to add the mandatory standardized header content to your source files.
