Lab #10 - Inheritance

Lab Introduction: Use Class Inheritance in C++

In this lab, you will apply your knowledge of class inheritance to complete two exercises that involve creating and using derived classes in C++.

Write functions and simple programs to practice and demonstrate your understanding of class inheritance, including creating derived classes, overriding member functions, and using base class pointers.

 

General Requirements

      Put each exercise in its own directory.
      Each class should be in its own source file (.cpp) and header file (.h).
      Use appropriate access specifiers (public, protected, private) for class members.
      Don't forget to add the mandatory standardized header content to your source files.
