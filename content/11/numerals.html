Numerals vs Numeric Data in C++

In C++, it is important to distinguish between numerals as characters or strings and numeric data types. While they may look similar, they are fundamentally different in how the compiler interprets and processes them.

  • Character: A single symbol, such as '5'
  • String: A sequence of characters, such as "123"
  • Numeric Type: A value used for arithmetic, such as 5 or 123
Characters: Numerals as Symbols

A character like '5' is not the number five. It is simply a symbol stored using its ASCII (or Unicode) value. 

#include <iostream>
using namespace std;

int main() {
    char c = '5';

    cout << c << endl;           // Outputs: 5
    cout << (int)c << endl;     // Outputs: 53 (ASCII value)

    return 0;
}

Even though '5' prints as 5, internally it is stored as the ASCII value 53.

Strings: Sequences of Numeral Characters

A string like "123" is a sequence of characters: '1', '2', and '3'. It is not automatically treated as the number 123. 

#include <iostream>
#include <string>
using namespace std;

int main() {
    string s = "123";

    cout << s << endl;   // Outputs: 123

    // This is NOT numeric addition
    cout << s + "1" << endl;  // Outputs: 1231

    return 0;
}

Concatenation occurs instead of arithmetic because strings store text, not numbers.

Numeric Types: Actual Values

Numeric types like int, double, etc. represent actual values that can be used in calculations. 

#include <iostream>
using namespace std;

int main() {
    int x = 123;

    cout << x + 1 << endl;  // Outputs: 124

    return 0;
}

Unlike strings, numeric types support arithmetic operations directly.

Converting Between Strings and Numbers

To use numeric values stored in strings, you must explicitly convert them. 

#include <iostream>
#include <string>
using namespace std;

int main() {
    string s = "123";

    int num = stoi(s);  // Convert string to integer

    cout << num + 1 << endl;  // Outputs: 124

    return 0;
}

Similarly, you can convert numbers to strings: 

int n = 456;
string str = to_string(n);

cout << str + "1" << endl;  // Outputs: 4561
Key Takeaways
  • '5' is a character, not the number 5
  • "123" is a string, not the number 123
  • 123 (without quotes) is numeric data
  • Strings concatenate; numbers perform arithmetic
  • Conversions like stoi() are required to treat strings as numbers
