Sorting Data in Arrays

Sorting means arranging data in a particular order, such as smallest to largest or alphabetically from A to Z. When working with arrays, sorting makes data easier to understand, search, and analyze.

For example, sorted data allows faster searching and produces cleaner output for users.

What Is an Algorithm?

An algorithm is a step-by-step set of instructions used to solve a problem or complete a task.

In computer science, algorithms tell the computer exactly how to process data. Sorting algorithms describe how values in an array should be compared and rearranged.

Sorting Algorithms

There are many common sorting strategies, each with different advantages and performance characteristics.

In this course, we start with simple algorithms that are easy to understand before moving on to faster, more advanced techniques.

Bubble Sort

Bubble Sort is one of the simplest sorting algorithms. It works by repeatedly comparing neighboring values and swapping them if they are in the wrong order.

Larger values gradually “bubble” toward the end of the array, which is how the algorithm gets its name.

How Bubble Sort Works
  1. Start at the beginning of the array
  2. Compare each pair of adjacent values
  3. Swap them if they are out of order
  4. Repeat until no more swaps are needed
Example Code (Bubble Sort)
// Bubble sort example
int numbers[5] = {5, 2, 4, 1, 3};

for (int pass = 0; pass < 4; pass++) {
    for (int i = 0; i < 4 - pass; i++) {
        if (numbers[i] > numbers[i + 1]) {
            int temp = numbers[i];
            numbers[i] = numbers[i + 1];
            numbers[i + 1] = temp;
        }
    }
}

Note: Bubble sort is easy to understand but can be slow for large arrays.

Check out the Wikipedia page on Bubble Sort for more details and visualizations of how it works.

Other Common Sorting Algorithms
  • Selection Sort: Repeatedly finds the smallest value and moves it to the correct position.
  • Insertion Sort: Builds a sorted portion of the array by inserting each new value into its proper position.
  • Merge Sort: Divides the array into smaller parts, sorts them, and merges them back together.
  • Quick Sort: Chooses a pivot value and rearranges elements around it, sorting recursively.

As you continue studying programming, you will see how different sorting algorithms trade simplicity for speed.
