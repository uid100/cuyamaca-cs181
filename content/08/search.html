Searching Through Arrays

In programming, searching means finding the location of a specific value inside a collection of data. Since arrays store multiple values under one name, we need efficient ways to search through them.

Here are two common search techniques:

  • Linear Search - checks each element one by one
  • Binary Search - repeatedly divides a sorted array in half
Linear Search

A linear search looks at each element in the array starting from the beginning and compares it to the value we are looking for.

This method works with any array, whether it is sorted or not.

How It Works
  1. Start at index 0
  2. Compare each value to the target
  3. Stop when the value is found or the array ends
Example Code (Linear Search)
// Search for a value in an array using linear search
int numbers[5] = {4, 7, 2, 9, 1};
int target = 9;
int index = -1;

for (int i = 0; i < 5; i++) {
    if (numbers[i] == target) {
        index = i;
        break;
    }
}

// index will be the position of target, or -1 if not found

Performance: Linear search may need to check every element in the worst case.

Binary Search

A binary search is much faster than linear search, but it only works on arrays that are sorted.

Instead of checking every value, binary search repeatedly cuts the search range in half.

How It Works
  1. Find the middle element
  2. If it matches, the search is complete
  3. If the target is smaller, search the left half
  4. If larger, search the right half
Example Code (Binary Search)
// Binary search on a sorted array
int numbers[6] = {2, 4, 6, 8, 10, 12};
int target = 8;

int low = 0;
int high = 5;
int index = -1;

while (low <= high) {
    int mid = (low + high) / 2;

    if (numbers[mid] == target) {
        index = mid;
        break;
    } else if (numbers[mid] < target) {
        low = mid + 1;
    } else {
        high = mid - 1;
    }
}

// index will be the position of target, or -1 if not found

Important: Binary search will give incorrect results if the array is not sorted.

Linear Search vs. Binary Search
  • Linear Search: simpler, works on any array, slower for large data
  • Binary Search: faster, but requires a sorted array

As arrays grow larger, choosing the right search method can significantly affect program performance.
