Programming Exercise: Caesar Cipher

Write a simple console-based C++ program that performs a Caesar cipher on a message entered by the user.

This is intended for practice with character arrays (or strings), functions, loops, and conditional logic.

What Is a Caesar Cipher?

A Caesar cipher is one of the earliest and simplest forms of encryption. It works by shifting each letter in a message forward or backward in the alphabet by a fixed number of positions.

For example, using a shift of 3:

  • A becomes D
  • B becomes E
  • X becomes A

When decoding, the process is reversed by shifting letters back.

Non-letter characters (spaces, punctuation, numbers) are typically left unchanged.

Program Behavior

Your program should:

  1. Ask the user whether they want to encode or decode a message
  2. Prompt the user to enter a message (up to 80 characters)
  3. Apply a Caesar cipher using a fixed shift value
  4. Display the resulting encoded or decoded message
  5. Use the constant value for the shift amount: const int SHIFT = 3;

The user may copy and paste longer sentences, but your program must limit the message size to a maximum of 80 characters.

Algorithm Outline
  1. Read the user choice (encode or decode)
  2. Read the message into a character array
  3. For each character in the message:
    • If it is a letter, apply the shift
    • Wrap around the alphabet if needed
    • Leave non-letter characters unchanged
  4. Output the transformed message

Program Requirements
  • Console application only (no GUI)
  • Maximum message length of 80 characters
  • Use constants for values such as message size and shift amount
  • Do not use encryption libraries
  • Ensure the program works for both uppercase and lowercase letters
Good Coding Standards & Reminders
  • Include the mandatory program header comment (name, course, assignment, date)
  • Use meaningful variable and function names
  • Break logic into functions where appropriate
  • Avoid magic numbers (unnamed constants). Use const variables such as MAX_MESSAGE_LENGTH and SHIFT.
  • Use consistent indentation and spacing
  • Comment your code to explain your logic
