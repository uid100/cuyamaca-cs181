Programming Assignment: Statistical Analysis

In this assignment, you will write a C++ program that performs simple statistical analysis on a set of integer values stored in an array.

Your program will calculate and display the following:

  • Minimum value
  • Maximum value
  • Mean (average)
  • Median

This assignment reinforces your understanding of arrays, functions, loops, and basic problem solving.

Sample Data

You may assume the program is given an array of integers similar to the following: 

int data[] = {12, 7, 3, 9, 15, 4};
const int SIZE = 6;

Do not change the values directly inside your functions. Functions should work with any integer array passed to them.

Statistical Concepts
Minimum and Maximum

The minimum is the smallest value in the array. The maximum is the largest value in the array.

Mean (Average)

The mean, also called the average, is calculated by adding all values in the array and then dividing by the number of values.

Median

The median is the middle value of a data set. Important: The data must be sorted before the median can be calculated.

  • If the number of values is odd, the median is the single middle value.
  • If the number of values is even, the median is the average of the two middle values.
Required Functions

You must write your own functions for each of the following tasks:

  • min() – returns the smallest value
  • max() – returns the largest value
  • mean() – returns the average
  • sort() – sorts the array in ascending order
  • median() – returns the median value

The median() function must assume the array has already been sorted.

Sample Output
Data values:
12 7 3 9 15 4

Minimum value: 3
Maximum value: 15
Mean: 8.3
Median: 8.0
Program Requirements
  • Use functions for all calculations
  • Do not use global variables
  • Do not use built-in sorting or statistics libraries
  • Use constants (e.g., SIZE) instead of hard-coded numbers
  • Display clear labels for all output
Good Coding Standards
  • Include the mandatory program header comment (name, course, assignment, date)
  • Use meaningful variable and function names
  • Indent code consistently
  • Avoid magic numbers (unnamed constants). Use const variables instead.
  • Write comments that explain why your code works

Clean, readable code is just as important as correct output.
