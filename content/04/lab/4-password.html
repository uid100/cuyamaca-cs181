Password Retries

Objective: Allow up to a maximum number of repeated login attempts using a do-while loop until the user enters the correct password, or the maximum number of attempts is reached.

Requirements

  • Name your file password_retry.cpp
  • Define the correct password as a constant (const char* correctPassword = "secret123";).
  • Prompt the user to enter a password.
  • Define a maximum number of login attempts const int MAX_ATTEMPTS = 3;.
  • Use a do-while loop to repeatedly prompt the user for the password.
  • If the user enters the correct password, print "Login successful."
  • If the user fails to enter the correct password within the maximum attempts, print "Login failed."

Sample Output

Enter password: hello
Incorrect password.

Enter password: test123
Incorrect password.

Enter password: secret123
Login successful.
