Number Guessing Game

Objective: Create a simple number guessing game where the user has to guess a randomly generated number between 1 and 100. The user will have a limited number of attempts to guess the correct number, and after each guess, the program will provide feedback on whether the guess is too low, too high, or correct.

Requirements

  • Name your file number_guessing.cpp
  • Define a maximum number of attempts const int MAX_ATTEMPTS = 5;.
  • Generate a random number between 1 and 100.
  • Seed the random number generator with srand(time(0));.
  • Use a constant seed const int SEED = 42; for testing. Comment this out, but leave it in your code.
  • Prompt the user to enter a guess.
  • Use a do-while loop to repeatedly prompt the user for guesses.
  • If the user enters the correct number, print "You guessed it!" and exit.
  • If the user fails to guess within the maximum attempts, print "Game over."
