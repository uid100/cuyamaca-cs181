What Is Whitespace?

In C++, whitespace includes:

  • spaces
  • tabs
  • newlines (Enter key)
  • carriage returns

The compiler mostly ignores whitespace, except when needed to separate identifiers and tokens.

Why Is Whitespace Important?

Whitespace impacts:

  • Readability - helps humans understand structure
  • Maintainability - reduces errors and makes debugging easier
  • Professionalism - teams expect clean formatting
  • Error Prevention - prevents merged tokens and ambiguous code

 

Have you ever read a text with no spaces? Hard, right?

Are There Rules?
Compiler Rules (Required)
intmain() { }  // ERROR: compiler thinks it's "intmain"
Style Rules (Strongly Recommended)
  • Indent code inside braces
  • Use consistent tabs or spaces
  • One statement per line
  • Use blank lines between sections

What Happens If You Don't Format Whitespace Well?

Poor whitespace can cause:

  • Hard-to-read code
  • Merged tokens and compiler errors
  • Hidden bugs
  • Team friction during reviews
