Introduction to Pointers

In C++, a pointer is a variable that stores the memory address of another variable. Instead of working directly with a value, pointers allow you to work with the location where that value is stored in memory.

Pointers are fundamental to understanding how C++ manages memory, enables efficient function calls, and supports dynamic data structures.

The Address Operator (&)

The address operator (&) is used to obtain the memory address of a variable. This operator does not change the variable’s value; it simply tells you where the variable is stored in memory. 


int number = 42;
int* ptr = &number; // ptr stores the address of number

In this example, &number produces the memory address of number, which is then stored in the pointer variable ptr.

Pointer Variables

A pointer variable is declared by placing an asterisk (*) between the data type and the pointer name. The data type indicates what kind of value the pointer will point to. 


int* intPtr;       // pointer to an int
double* dblPtr;   // pointer to a double
char* charPtr;    // pointer to a character

The asterisk is part of the declaration and means “pointer to.” It does not dereference the pointer at this stage.

Declaring and Assigning Pointer Values

Pointers should be assigned a valid address before being used. This is typically done using the address operator. 


int value = 10;
int* ptr = &value;  // ptr points to value

To access or modify the value a pointer points to, use the dereference operator (*). 


*ptr = 20;          // changes value to 20
std::cout << value; // outputs 20

Dereferencing tells the program to “go to the address stored in the pointer and work with the value found there.”

Passing Pointers to Functions

Pointers are commonly passed to functions when the function needs to modify the original variable or work efficiently with large data. 


void updateValue(int* p) {
    *p = 100;
}

int main() {
    int num = 5;
    updateValue(&num);
    // num is now 100
}

Here, the function receives a pointer to num. By dereferencing that pointer inside the function, the original variable in main is changed.

Key Takeaways

  • Pointers store memory addresses, not actual values.
  • The address operator (&) gets a variable’s address.
  • The dereference operator (*) accesses the value at an address.
  • Passing pointers to functions allows direct modification of arguments.
