Lab Exercise 2: Swapping Values Using Pointers

In this exercise, write a C++ program that uses pointers to modify variables passed to a function. This demonstrates how pointers allow functions to directly change values stored in memory.

Learning Goal

Practice passing pointers to functions and using dereferencing to modify the original variables.

Program Requirements

Your program must do the following:

  • Define a function that accepts two int* parameters
  • Swap the values stored at the addresses provided
  • Declare two integer variables in main
  • Call the swap function using the addresses of those variables
  • Display the values before and after the function call
