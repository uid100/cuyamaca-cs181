Lab Exercise 1: Pointer Basics and Dereferencing

In this lab, you will write a simple C++ program to explore how pointers work. You will practice using the address-of operator (&) and the dereference operator (*).

Learning Goal

Understand how pointers store memory addresses and how to access a variable’s value indirectly using pointer notation.

Program Requirements

Your program must do the following:

  • Declare an int variable and assign it a value
  • Declare a pointer that points to the int variable
  • Print the value of the variable
  • Print the address of the variable
  • Print the value accessed through the pointer

Starter Code

Use the following code as a starting point. Fill in the missing parts as needed. 


#include <iostream>
using namespace std;

int main() {
    int value = 0;     // declare and initialize an int variable
    int* ptr = nullptr; // declare a pointer

    // assign the pointer to the address of value

    // print the value of value

    // print the address of value

    // print the value accessed through ptr

    return 0;
}

Expected Behavior

When the program runs, it should display:

  • The integer value stored in the variable
  • The memory address of that variable
  • The same integer value accessed through the pointer

The exact memory address will vary each time the program runs.

Submission Checklist

  • Your code compiles without errors
  • You correctly use & to get an address
  • You correctly use * to dereference a pointer
  • Your output clearly labels each printed value
