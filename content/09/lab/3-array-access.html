Lab Exercise 3: Array Access Using Pointer Notation

This time, use pointers and pointer arithmetic to access and display elements of an array. This exercise demonstrates the close relationship between arrays and pointers in C++.

Learning Goal

Understand how arrays are stored in memory and how pointer notation can be used to access array elements without using array indexing.

Program Requirements

Your program must meet the following requirements:

  • Declare and initialize an array of integers
  • Use a pointer to refer to the first element of the array
  • Print all array elements using pointer dereferencing
  • Use pointer arithmetic to move through the array
  • Do not use the array indexing operator ([])

Starter Code

Use the following code as a starting point. Complete the loop so that it prints each element using pointer notation. 


#include <iostream>
using namespace std;

int main() {
    int numbers[5] = {10, 20, 30, 40, 50};
    int* ptr = numbers;   // points to the first element

    // use a loop to print all elements using ptr
    // do NOT use array indexing []

    return 0;
}

Expected Behavior

When the program runs, it should print each value stored in the array in order. The output should match the values in the array.

Pointer arithmetic should be used to move from one element to the next.

Submission Checklist

  • Your program compiles without errors
  • You use a pointer to access array elements
  • You use pointer arithmetic (ptr + i or pointer incrementing)
  • You do not use [] to access array elements
  • Your output clearly displays all array values
