C++ Exercise: Class Inheritance (Animal → Dog)
Objective

Practice basic class inheritance, method reuse, and function overriding by implementing an Animal base class and a Dog derived class.

Project Requirements
  • Create a base class named Animal.
  • Add one public member function:
    • void speak() that outputs exactly: "Animal makes a sound"
  • Create a derived class named Dog that:
    • Uses public inheritance from Animal
  • In the Dog class:
    • Define a new version of void speak()
    • This function must override the base class version
    • Output exactly: "Dog barks"
  • In the main() function:
    • Create a Dog object
    • Call speak() using the Dog object
  • Demonstrate method reuse:
    • Call the original Animal::speak() function from the Dog object
    • Use scope resolution (DogObject.Animal::speak())
  • Include required elements:
    • #include <iostream>
    • using namespace std;
Program Output Requirements

Your program must produce the following output (order matters):

Dog barks
Animal makes a sound
Constraints
  • No global variables
  • No additional classes beyond Animal and Dog
  • Output must match exactly (including capitalization and spacing)
Key Concepts Demonstrated
  • Inheritance: Dog inherits from Animal
  • Method Reuse: Calling base class function from derived class object
  • Function Overriding: Redefining speak() in Dog
Optional Extension
  • Add a second derived class (e.g., Cat) with its own speak()
  • Test all classes in main()
