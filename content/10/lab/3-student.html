C++ Exercise: Multi-Level Inheritance (Person → Student → GraduateStudent)
Objective

Implement a multi-level inheritance chain in C++ to demonstrate how data and behavior are passed through multiple derived classes.

Project Requirements
  • Create a base class named Person.
    • Include one public data member: string name
  • Create a class named Student that:
    • Inherits publicly from Person
    • Adds one public data member: int id
  • Create a class named GraduateStudent that:
    • Inherits publicly from Student
    • Adds one public data member: string thesis
  • In the GraduateStudent class:
    • Create a public member function void display()
    • This function must output all three values:
      • Name (inherited from Person)
      • ID (inherited from Student)
      • Thesis topic (defined in GraduateStudent)
  • In the main() function:
    • Create one GraduateStudent object
    • Assign values directly to all inherited and declared members:
      • name
      • id
      • thesis
    • Call the display() function
  • Program must include:
    • #include <iostream>
    • #include <string>
    • using namespace std;
Output Requirements

Your program must print all three fields clearly in a single function call. Follow a readable format such as: 

Alice, ID: 101, Thesis: AI Research
Implementation Notes
  • Do not duplicate variables across classes
  • Access inherited members directly from the most derived class
  • Ensure proper use of public inheritance at each level
Minimal Example Structure (Guidance Only)
// Class declarations only (details omitted)
class Person {
  // name
};

class Student : public Person {
  // id
};

class GraduateStudent : public Student {
  // thesis + display()
};
Key Concepts Demonstrated
  • Multi-level inheritance: A class derived from another derived class
  • Data inheritance: Child classes access parent class members
  • Code organization: Extending functionality step-by-step
Optional Challenge
  • Add a constructor to each class and initialize values using constructor chaining
  • Create a second GraduateStudent and display both
