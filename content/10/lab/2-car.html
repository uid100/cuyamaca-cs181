C++ Exercise: Inheritance with Constructors (Vehicle → Car)
Objective

Demonstrate how constructors are inherited and called in C++ by creating a base class Vehicle and a derived class Car.

Project Requirements
  • Create a base class named Vehicle.
    • Include one public data member: string brand
    • Define a constructor that:
      • Accepts one parameter (string)
      • Initializes the brand member
  • Create a derived class named Car.
    • Use public inheritance from Vehicle
    • Add one public data member: string model
  • In the Car class:
    • Create a constructor that:
      • Accepts two parameters: brand and model
      • Calls the base class (Vehicle) constructor using an initializer list
      • Initializes the model member
  • Add a member function in Car:
    • void display()
    • Outputs both brand and model on one line
  • In the main() function:
    • Create one Car object
    • Pass values for both brand and model through the constructor
    • Call the display() function
  • Program must include:
    • #include <iostream>
    • #include <string>
    • using namespace std;
Output Requirements

Your program must display both values clearly, for example:

Toyota Corolla
Implementation Notes
  • You must use an initializer list to call the base class constructor
  • Do not assign the brand value inside the Car constructor body
  • The Car constructor must explicitly call Vehicle(brand)
Minimal Starter Code
#include <iostream>
#include <string>
using namespace std;

class Vehicle {
public:
    string brand;

    // constructor here
};

class Car : public Vehicle {
public:
    string model;

    // constructor here (must call Vehicle constructor)

    void display();
};

int main() {
    // create Car object and test
}
Key Concepts Demonstrated
  • Constructor inheritance: Derived class uses base constructor
  • Initializer lists: Required to call parent constructors
  • Code reuse: Base class handles shared data initialization
Optional Challenge
