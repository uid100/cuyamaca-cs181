At the end of this module, you will be able to:

  • Write functions to modularize code
  • Understand variable scope and lifetime
  • Pass parameters by value or reference
  • Overload functions with different signatures

In this module, we will explore the concepts of functions and scope in C++ and how writing modular code using functions can improve code organization and reusability. We will also discuss variable scope and lifetime, and how to pass parameters by value or reference. Finally, we will look at function overloading and how it allows us to define multiple functions with the same name but different parameter lists.
