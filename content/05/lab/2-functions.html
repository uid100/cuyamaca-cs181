Lab Instructions: Writing Custom C++ Functions

Overview

In this assignment, you will write a simple C++ program that defines and calls three different custom functions:

  • One function that returns an integer
  • One function that returns a double
  • One function that returns void

The completed program will demonstrate the differences between value-returning functions and procedures, as well as the basics of passing parameters by value.

File & Structure Requirements
  • Your program must be written in a single file named YourProgramName.cpp.
  • Include the standard header content used in all your previous assignments.
  • Follow proper naming conventions for variables and functions.
  • Indent consistently and use clear whitespace formatting.
  • Your function prototypes must appear above main().
  • Your function definitions must appear after main().
  • Each function prototype must include parameter names, not just data types.
Function Requirements

Your program must define and use the following types of functions:

  • Integer-returning function
    – Should perform a simple calculation or return a meaningful integer value.
  • Double-returning function
    – Should return a double value based on one or more parameters.
  • Void function
    – Should perform an action such as displaying output, with no return value.

Each function must accept at least one parameter passed by value.

Program Behavior
  • main() must call each of your custom functions at least once.
  • All output must be clear, labeled, and easy for the user to understand.
  • Functions must be tested with appropriate sample input values.
  • The program must compile and run without warnings or errors.
What to Submit
  • Your full .cpp file with header content, main(), prototypes, and function definitions.
  • Ensure all formatting and naming conventions follow class standards.
