Variable Scope and Lifetime

In C++, scope refers to where a variable can be accessed in a program. Lifetime refers to how long that variable exists in memory.

Types of scope:

  • Local scope — variables declared inside a function or block; accessible only there.
  • Global scope — variables declared outside all functions; accessible throughout the file.

Local variables exist only while the function is running. Global variables exist for the entire duration of the program.

Passing Values as Parameters

Parameters allow us to send information into a function so it can work with different data. Parameters behave like local variables inside the function.

In introductory C++, we focus on pass‑by‑value, where the function receives a copy of each argument. Changing the parameter inside the function does not change the original value in main(). 

void showDouble(int x) {
    cout << x * 2;
}

Returning a Value from a Function

When a function needs to send a result back to the place where it was called, it uses a return statement. The returned value must match the function’s return type. 

int add(int a, int b) {
    return a + b;   // sends the sum back to the caller
}

A function stops running as soon as it reaches a return statement.

Local and Global Variables

Local variables are declared inside a block or function and exist only while that function or block is running.

Global variables are declared outside all functions. They exist for the entire runtime of the program and are accessible everywhere in the file.

In good practice, global constants are preferred over global variables. Constants are safe because their values cannot be changed. 

const double PI = 3.14159;  // global constant

Use global constants when multiple functions need to share the same fixed value.

Local Static Variables

A local static variable is created inside a function but behaves differently from a regular local variable.

  • It is created only once.
  • It keeps its value between function calls.
  • Its scope is still local to the function.
void counter() {
    static int count = 0;  // initialized only once
    count++;
    cout << count;
}

Every time counter() is called, count continues increasing— it does not reset. This is useful for tracking how many times a function has run.
