Tracing Program Execution

Tracing is the process of following the order in which statements and function calls are executed in a program. This helps students visualize how values move through functions, how the call stack works, and why functions return control back to their caller.

The example below shows a main function that calls two other functions, and one of those functions calls a third helper function. We will trace the execution line‑by‑line to understand the flow.

Sample C++ Code (with Line Numbers)


 1 | #include <iostream>
 2 | using namespace std;
 3 |
 4 | int add(int, int);
 5 | void greet();
 6 | int doubleIt(int);
 7 |
 8 | int main() {
 9 |     cout << "Program starting..." << endl;
10 |     greet();
11 |     int value = add(3, 4);
12 |     cout << "Result from add: " << value << endl;
13 |     return 0;
14 | }
15 |
16 | void greet() {
17 |     cout << "Hello from greet()" << endl;
18 |     int doubled = doubleIt(5);
19 |     cout << "Doubled value: " << doubled << endl;
20 | }
21 |
22 | int add(int a, int b) {
23 |     return a + b;
24 | }
25 |
26 | int doubleIt(int x) {
27 |     return x * 2;
28 | }

Tracing Execution Step-by-Step

1. Program starts at line 8: main() begins.

2. Line 9: Prints: "Program starting..."

3. Line 10: greet() is called. Execution jumps to line 16.

4. Line 17: Prints: "Hello from greet()"

5. Line 18: doubleIt(5) is called. Execution jumps to line 26.

6. Line 27: x * 2 evaluates to 10. return 10; sends this value back to greet().

7. Line 19: doubled now contains 10. Prints: "Doubled value: 10"

8. End of greet() at line 20: Control returns to main(), line 11.

9. Line 11: add(3, 4) is called. Execution jumps to line 22.

10. Line 23: a + b evaluates to 7. Returns 7 to main().

11. Line 12: Prints: "Result from add: 7"

12. Line 13: return 0; ends the program.

The final output (in order) would be:


Program starting...
Hello from greet()
Doubled value: 10
Result from add: 7
