At the end of this module, you will be able to:

  • Write and use custom classes
  • Understand the file structure for C++ classes
  • Define and use struct in C++

In this module, we will explore the concepts of classes and objects in C++ and how writing modular code using classes can improve code organization and reusability. We will also discuss the lifecycle of objects, and how to pass objects to functions. Finally, we will look at object composition and how it allows us to define complex objects from simpler ones.
