Tic Tac Toe

Tic Tac Toe is a simple game played on a 3x3 grid. Two players take turns marking a space in the grid with their respective symbols (usually X and O). The first player to get three of their symbols in a row (horizontally, vertically, or diagonally) wins the game. If all spaces are filled and no player has three in a row, the game is a draw.

Overview

In this console-based C++ exercise, the computer plays both X and O in a Tic-Tac-Toe game. There is no user input. The goal is to practice working with 2D arrays, loops, random numbers, and functions.

Requirements

  • Use a 3 x 3 2D array of characters to represent the board.
  • Initialize all spaces to '-' to indicate blank spaces.
  • The computer alternates turns between 'X' and 'O'.
  • For each turn, randomly select a row and column.
  • If the selected space is already taken, generate new random values until an empty space is found.
  • After each valid move:
    • Print the updated grid
    • Check for a winner
  • If a player gets three in a row (row, column, or diagonal), display the winner and stop the game.
  • If the board fills with no winner, declare a tie.

Turns should be randomly generated, no user input or AI logic is required or expected.

Simplified Game Logic

  1. Create and initialize the board.
  2. Set the current player to 'X'.
  3. Repeat until the game ends:
      Display Player X or Player O.
      Generate a random row and column.
      If the space is empty, place the player's symbol there.
      Print the board.
      Check for a winner.
      If no winner and the board is not full, switch players.

Functions to Implement

  • void initializeBoard(char board[3][3]);
  • void printBoard(char board[3][3]);
  • bool checkWinner(char board[3][3], char player);
  • bool isBoardFull(char board[3][3]);
