Array Utilities

In this lab, you will practice creating and using arrays in C++.

You will create a set of utility functions that operate on arrays, and have a look at automated testing to verify your implementations.

Incremental Development

Do not "write everything at once". Instead, work step-by-step through the instructions. Each TODO unlocks the next test. This is a common approach to incremental development and testing.

Separation of Concerns

Add the following three files to your lab folder:

  • Tests live in ArrayTests.cpp
  • Function prototypes live in ArrayFunctions.h
  • Function implementations live in ArrayFunctions.cpp

This mirrors professional C++ workflows.

Important: Each file must begin with the mandatory standard file header content required by this course.

Automated Testing

  • Code correctness is verified automatically
  • Passing tests = correct behavior
  • Failing tests = bugs to fix
  • Regression is visible immediately

Build Fundamentals

The project:

  • Compiles and runs even with incomplete implementations
  • Has no hard-coded data in the header or implementation files
  • Forces correct linkage of multiple .cpp files
  • Reinforces why headers don't produce code

Note: If your function is correct, the test will pass without modification.
If the test fails, debug your function — not the test.
