Lab Introduction: Create and Use Arrays in C++

Overview

Next, you will apply your knowledge of arrays and structs to complete two exercises that involve creating and using arrays in C++.

In the first, you will implement a set of utility functions that operate on arrays, and have a look at automated testing to verify your implementations.

In the second, you will create a simple console-based Tic Tac Toe game that uses a 2D array to represent the game board and implements the game logic.

Instructions

Be sure to follow all instructions carefully and include the required standard file header content in each source file.

Where code is provided, there are sections marked with "TODO" comments that indicate where you need to add code to complete the implementation. You should not modify any of the provided code outside of these sections. When the tasks are complete, edit the file to remove the "TODO" comments and add comments to explain your code as needed.
