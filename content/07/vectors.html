Vectors

A vector is a dynamic array that can grow and shrink in size. It is part of the C++ Standard Template Library and provides a convenient way to store and manage collections of data.

Vectors are implemented as templates, which means you can create vectors of any data type. They provide various member functions for adding, removing, and accessing elements, as well as for resizing the vector.

The textbook provides a brief introduction to vectors and we'll discuss these data structures briefly in class. Vectors will be covered in more depth in your data structures course. For now, you should understand that vectors are a powerful and flexible alternative to arrays, especially when you need a collection that can change in size.

Vector Member Functions

Some of the commonly used member functions of vectors include:

  • push_back(): Adds an element to the end of the vector.
  • pop_back(): Removes the last element from the vector.
  • size(): Returns the number of elements in the vector.
  • clear(): Removes all elements from the vector.
  • resize(): Changes the size of the vector.
  • at(): Accesses an element at a specific index with bounds checking.
  • swap(): Exchanges the contents of two vectors.
