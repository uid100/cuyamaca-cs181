What Is a PRNG?

Pseudo-Random Number Generator

A PRNG is a math‑based generator that produces repeatable sequences of numbers that only appear random.

PRNG stands for Pseudo‑Random Number Generator. It is a program that produces numbers that look random, but are actually generated using mathematical rules and algorithms.

Why it’s called “pseudo-” random

PRNGs are not truly random. If you start them with the same starting value (called a seed), they will always produce the same sequence of numbers.

This is useful for testing and debugging programs, because you can get the same results each time you run the program.

True randomness requires physical sources, such as:

  • Mouse movements
  • Keyboard timing
  • CPU temperature noise
  • Hardware random-number generators

PRNGs in C++

In C++, the function rand() uses a PRNG. The function srand() sets the starting value (the seed). 


        srand(42);  // set seed to 42
        rand();  // same result every run
        rand();

Using a different seed produces a different sequence: 


        srand(100);
        rand();  // different sequence
        rand();

Why this matters

  • Same seed → same results (good for testing)
  • Different seed → different results
  • Without srand, the sequence is always the same each run

 

Important note

PRNGs are fine for simple programs, games, and class assignments, but they should not be used for security, passwords, or encryption.
