Program: Fahrenheit to Celsius Calculator

Write a C++ program that reads a temperature in Fahrenheit from the user (accepting either an integer or a double), converts it to Celsius using named constants, and displays the result.

Specifications:

File name:
fahrenheit_to_celsius.cpp
Focus:
Use input/output with cin/cout, arithmetic expressions with named constants, and clear formatted output.
Constants (use these; do not hard‑code numbers inline):
const double OFFSET = 32.0;     // the 32° offset between scales
const double NUMERATOR = 5.0;   // 5 part of 5/9
const double DENOMINATOR = 9.0; // 9 part of 5/9
Formula:
Celsius = (Fahrenheit - OFFSET) * (NUMERATOR / DENOMINATOR)
Input:
One value entered by the user as Fahrenheit (e.g., 32, 98.6, or -40).
Output:
A clearly labeled Celsius value computed from the input Fahrenheit.
Example:
Enter temperature in Fahrenheit: 98.6
Celsius: 37.0
  • Include #include <iostream> and use double variables for the Fahrenheit input and Celsius result so the code works for both integer and floating‑point entries.
  • Avoid unnamed constants: do not write numbers like 32 or 5.0/9.0 directly in the expression—use the named constants provided.
  • Compute using (fahrenheit - OFFSET) * (NUMERATOR / DENOMINATOR); the division will be floating‑point because the constants are double.
  • Print the result with a label (e.g., "Celsius: "), and consider fixed formatting (e.g., one decimal place) if desired.
  • No loops, no conditionals, and no arrays; keep it to input, arithmetic, and output.
  • Include the required file header comment block as discussed in class and ensure it compiles from the command line.
