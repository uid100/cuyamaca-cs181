Program: Greeting with a C‑String

Write a C++ program that reads a user’s first name into a C‑string and prints a greeting message using that name.

Specifications:

File name:
cstring_greeting.cpp
Focus:
Use C‑strings (char[]), input with cin, and output with cout.
Input:
The user’s first name (a single word with no spaces).
Output:
A greeting message that includes the entered name.
Example:
Enter your first name: Alex
Hello, Alex!
  • Declare a char array large enough to hold the name and the null terminator (e.g., char name[20];).
  • Use cin >> name; to read the name into the C‑string.
  • This is a C‑string, not a std::string: do not use string objects or assignment.
  • The input will stop at whitespace, so this program only needs to handle a first name.
  • No loops, no conditionals, and no arrays other than the C‑string.
  • Include the required file header comment block and ensure the program compiles from the command line.
