Program: Average of Three Test Scores

Write a C++ program that gets and stores the values of three test scores as integers, calculates their average as a floating‑point number, and displays the result.

Specifications:

File name:
average_three_scores.cpp
Focus:
Use int for inputs, compute a floating‑point average with type casting, and produce clear labeled output.
Constants (avoid magic numbers):
const int NUM_TESTS = 3; // Use this instead of writing "3" in multiple places
Input:
Three integer test scores entered by the user.
Output:
The average of the three scores as a floating‑point number (e.g., with 1–2 decimal places).
Example:
Enter score 1: 90
Enter score 2: 86
Enter score 3: 95
Average: 90.33
  • Include #include <iostream> and store each score in an int variable.
  • Avoid unnamed constants (magic numbers): use NUM_TESTS when dividing to compute the average.
  • Compute the average as floating‑point by casting the sum or one operand to double (e.g., static_cast<double>(sum) / NUM_TESTS).
  • Optionally use formatted output (e.g., std::fixed and std::setprecision(2)) for readability.
  • No loops, no conditionals, and no arrays; read exactly three scores and compute directly.
  • Include the required file header comment block as discussed in class and ensure it compiles from the command line.
