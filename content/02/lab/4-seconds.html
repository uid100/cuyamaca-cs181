Program: Seed PRNG and Convert Seconds

Write a C++ program that prompts the user for an integer seed, initializes the pseudo‑random number generator, takes the first value returned by rand() as a number of seconds, and converts it into weeks, days, hours, minutes, and seconds, then displays the result.

Specifications:

File name:
seed_seconds_convert.cpp
Focus:
Use std::srand/std::rand, integer arithmetic (division and modulus), constants, input, and output.
Input:
One integer seed entered by the user.
Output:
The random seconds value in a labeled breakdown of weeks, days, hours, minutes, and seconds.
Example:
Enter seed: 1
(16807s) = 0 weeks, 0 days, 4 hours, 40 minutes, 7 seconds
  • Use #include <cstdlib> for std::srand/std::rand and #include <iostream> for input/output.
  • Call std::srand(seed) with the user’s integer, then call std::rand() once to get the seconds value.
  • Define clear constants, e.g., const int SECS_PER_MIN = 60;, SECS_PER_HOUR = 3600;, SECS_PER_DAY = 86400;, SECS_PER_WEEK = 604800;.
  • Hint (modulus): Use integer division and modulus to peel off each unit:
    • weeks = total / SECS_PER_WEEK;
    • rem = total % SECS_PER_WEEK; (remainder after weeks)
    • days = rem / SECS_PER_DAY;, then rem = rem % SECS_PER_DAY;
    • hours = rem / SECS_PER_HOUR;, then rem = rem % SECS_PER_HOUR;
    • minutes = rem / SECS_PER_MIN;, and seconds = rem % SECS_PER_MIN;
    Think: “divide to get the count, use % to keep what’s left for the next smaller unit.”
  • No loops, no conditionals, and no arrays; just arithmetic and straightforward calculations.
  • Format your output similar to the example; include the original seconds in parentheses with an s suffix.
  • Include the required file header comment block as discussed in class and ensure it compiles from the command line.
