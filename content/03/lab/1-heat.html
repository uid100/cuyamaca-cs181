Temperature Checker
if / else if / else (C++)

Objective: Read an integer Fahrenheit temperature from the user and classify it as cold, warm, or hot using an if / else if / else structure. Keep everything in main(), include the standard header comment from earlier labs, and avoid magic numbers by using named constants.

Requirements

  • Input: One integer temperature in Fahrenheit (e.g., -5, 0, 72, 100).
  • Classification: Print exactly one of: "cold", "warm", or "hot".
  • Decision logic: Use a clear if / else if / else chain.
  • No magic numbers: Define named constants (e.g., COLD_MAX_F, HOT_MIN_F) and use those in comparisons.
  • One function only: All code resides in main(). (No functions/methods yet.)
  • Header block: Include the standard header comment used in earlier labs at the top of the .cpp file.
Tip: Choose and document your thresholds with named constants at the top of main(). For example, you might define a maximum for “cold” and a minimum for “hot”; “warm” is the range in between.

Suggested Behavior

  • If the temperature is at or below your “cold max” constant → print cold.
  • Else if the temperature is at or above your “hot min” constant → print hot.
  • Else → print warm.
