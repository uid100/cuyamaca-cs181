C++ Assignment: Grade Calculator

Write a program that collects multiple assignment scores from the user, calculates the average of those scores, and displays the corresponding letter grade. This program must follow the structural and style guidelines used in previous labs.

Program Requirements

  • Include the **standard header block** at the top of your .cpp file (same format used in earlier labs).
  • All logic must be contained within main(). Do not write additional functions or methods.
  • Use a **loop** to collect assignment scores as floating-point percentages (e.g., 87.5).
  • The loop must continue **as long as the user enters input that can be converted to a double**.
  • When the user enters anything that is not a valid floating-point number, the loop ends.
  • Keep a **running total** of all valid scores and maintain a **count** of how many scores were entered.
  • After the loop ends, calculate and display the **average score**.
  • Use an if - else if - else structure to convert the numeric average into a letter grade:
    • average ≥ 90 → A
    • average ≥ 80 → B
    • average ≥ 70 → C
    • average ≥ 60 → D
    • otherwise → F
  • No magic numbers: All numeric grade thresholds must be named constants declared at the top of main().
  • After displaying the average and letter grade, the program should end normally.
Input behavior:
Prompt the user before each score. If they type a number (e.g., 91 or 72.4), it is added to the running total. If they type anything else (e.g., x, done, or blank input), the loop ends.

Output Requirements

  • Display the calculated average score.
  • Display the letter grade determined by your if / else if / else logic.
  • Output should be clear and readable.

Before You Submit

  • Your header block is complete and correctly formatted.
  • Your program compiles and runs without warnings.
  • All values used in conditions are named constants.
  • Loop collects scores until the user enters invalid numeric input.
  • You use exactly one if / else if / else chain for letter-grade determination.

Additional notes

  • Output does not need to be rounded or printed with a specified precision.
  • Only positive values, between 0 and 100 should be considered valid scores.
  • Zero points is a valid score and should be added to the total and increment the count.
