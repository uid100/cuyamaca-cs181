C++: Random Sales Price & Make Change (Fewest Coins)

Beginner instructions — all logic in main(), include your standard header, and avoid unnamed constants.

Goal
  • Generate a random sales price < $1.00.
  • Assume the customer pays with $1.00; compute the change in the fewest coins using quarters, dimes, nickels, pennies.
  • Use a clear if / else if / else structure with simple arithmetic (no functions, keep everything in main()).
  • No magic numbers: define named constants for coin values and for $1.00 in cents.
  • Display the random price and the coin breakdown.
Program Requirements
  • Include the required standard header block (from earlier labs) at the very top of your .cpp file.
  • All logic in main(); do not write additional functions or methods.
  • Use named constants for: ONE_DOLLAR_CENTS, QUARTER, DIME, NICKEL, PENNY.
  • Generate a random price under $1.00 and print it as dollars.cents (e.g., $0.73).
  • Compute change from $1.00, then use if / else if / else with simple subtraction to form the fewest coins (try larger coins first).
Hint: Generating a Random Price (using std::srand / std::rand)

Seed once with the current time, then bound the random value to the range [0.00, 0.99]. Work in cents to avoid rounding errors, then format for display.

#include <cstdlib>   // std::srand, std::rand
#include <ctime>     // std::time

std::srand(static_cast(std::time(nullptr)));
int priceCents = std::rand() % 100;     // 0..99 cents  (i.e., $0.00 .. $0.99)
// Optional as double between 0.00 and 0.99:
// double price = static_cast(priceCents) / 100.0;

Display tip: show as dollars.cents with leading zeroes for cents (e.g., 7 → "$0.07").

Hint: Control Flow Skeleton (If / Else‑If / Else)

Start with a remaining change variable (in cents). Subtract the largest coin while possible, then move to the next coin size.

// Named constants at the top of main(): ONE_DOLLAR_CENTS=100, QUARTER=25, DIME=10, NICKEL=5, PENNY=1
int change = ONE_DOLLAR_CENTS - priceCents;
int quarters = 0, dimes = 0, nickels = 0, pennies = 0;


while (change > 0) {
    if (change >= QUARTER) {
        change -= QUARTER; 
        quarters++;
    } else if (change >= DIME) {
        change -= DIME; 
        dimes++;
    } else if (change >= NICKEL) {
        change -= NICKEL; 
        nickels++;
    } else { // pennies
        change -= PENNY; 
        pennies++;
    }
}

Print the original price and the counts for each coin type.

Hint: Format Output

Use iomanip to show two decimal places and a leading zero for cents. If you store the price in cents, either divide to a double and format, or print dollars and cents separately.

#include <iomanip>  // std::fixed, std::setprecision, std::setfill, std::setw

// Option A: using a double
double price = static_cast(priceCents) / 100.0;
std::cout << "$" << std::fixed << std::setprecision(2) << price; // $0.73

// Option B: printing dollars and cents separately from ints
int dollars = priceCents / 100;
int cents   = priceCents % 100;
std::cout << "$" << dollars
          << "." << std::setw(2) << std::setfill('0') << cents; // $0.73

Apply the same approach when displaying totals or other monetary values.
