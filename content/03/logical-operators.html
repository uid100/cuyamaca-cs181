Boolean Variables and Boolean Expressions

A boolean variable is a data type that can hold one of two values: true or false.

Examples:


    #include <iostream>
    using namespace std;
    int main() {
        bool isRaining = true;
        bool isSunny = false;
        cout << "Is it raining? " << isRaining << endl; // true
        cout << "Is it sunny? " << isSunny << endl;   // false
        return 0;
    }

A boolean expression is an expression that evaluates to a boolean value (true or false). Examples of boolean expressions include comparisons like 5 > 3 (which evaluates to true) and 2 == 4 (which evaluates to false).

The distinction is important. A boolean variable will always hold a boolean value. With a relational operator, any variable type may be used to create a boolean expression. 

int heightOfTreeInFeet = 12; // integer variable
if (heightOfTreeInFeet > 10) { // boolean expression using relational operator
    cout << "The tree is tall!" << endl; // executes because expression is true
}

Like a boolean variable, a boolean expression will always evaluate to either true or false.

Logical Operators in C++

In C++, logical operators are used to combine multiple boolean expressions. The primary logical operators are:

  • AND (&&): Returns true if both operands are true.
  • OR (||): Returns true if at least one operand is true.
  • NOT (!): Returns true if the operand is false.

Examples:


#include <iostream>
using namespace std;
int main() {
    bool a = true;
    bool b = false;
    cout << "a AND b: " << (a && b) << endl; // false
    cout << "a OR b: " << (a || b) << endl;  // true
    cout << "NOT a: " << (!a) << endl;      // false
    return 0;
}

Logical operators are commonly used in conditional statements to control the flow of a program based on multiple conditions.

Best practice suggests positioning the more complex condition on the right side of an AND (&&) operator to optimize performance, as the left condition is evaluated first and may short-circuit the evaluation.
