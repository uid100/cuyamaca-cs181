At the end of this module, you will be able to:

  • Explain the use and purpose of boolean expressions in decision making
  • Use relational and logical operators to create boolean expressions
  • Implement if, if/else, and if/else if statements to control program flow
  • Use nested if statements to implement complex decision structures
  • Validate user input to ensure data integrity
  • Use the switch statement to implement multi-way branching

Chapter 4 deals with logical decision making in C++. Many of the concepts presented here will be familiar to you if you have experience with other programming languages. However, the syntax and specific implementation details will be unique to C++.

We will start with a discussion of boolean expressions and relational operators, which are the building blocks of decision making in programming. We will then explore the various forms of decision control structures, such as the if statement in different variations, and the switch statement and practical applications such as menu-driven programs.
