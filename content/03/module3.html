At the end of this module, you will be able to:

  • Understand declaring and using Variables in C++
  • Understand primitive data types in C++
  • Understand and use arithmetic expressions and operators
  • Use the cin and cout objects for input and output
  • Understand and apply style and naming conventions
  • Apply and follow mathematical order of operations
  • Use the C++ random number generator

Chapter 3 is titled Expressions and Interactivity and it is full of important concepts and information that you will need to understand in order to be successful C++ programmers.

Although many of the concepts will be familiar to you from other programming languages, the author does a great job of explaining not only how things work in C++, but also why they work that way. Be prepared to spend some time in this chapter.

As always, you should try the examples on your own while you read the content. Experiment with the code to see how changes affect the output. This is a great way to learn!
