Assignment: Estimating π Using Structs and Monte Carlo Simulation

This assignment demonstrates how structs, functions, random number generation, and loops can work together to solve a mathematical problem.

You will use the Monte Carlo method to estimate the value of π by randomly generating points and determining how many fall inside a circle.

Learning Objectives

  • Define and use a struct with data and behavior
  • Generate random numbers in C++
  • Use functions to return structured data
  • Apply a Monte Carlo simulation
  • Compute an approximation and error (variance)

File Requirements

Build this assignment as a single C++ file named:

  • EstimatePi.cpp

The mandatory standard file header must appear at the top of the file.

Step 1: Define the Point Struct

Use the Point struct shown below. This struct stores an x and y coordinate and also includes a member function that calculates the distance from the origin. 


struct Point
{
    double x;
    double y;

    double distanceFromOrigin() const
    {
        return std::sqrt(x * x + y * y);
    }
};

This demonstrates an important idea: although structs are often used to group data, they can also contain functions.

Step 2: Understanding the Monte Carlo Method

To estimate π, imagine a circle of radius 1 centered at the origin, inside a square that runs from -1 to +1 on both axes.

  • Randomly generate points within the square
  • Check whether each point is inside the circle
  • The ratio of points inside the circle to total points approximates π / 4

From this ratio, you can compute an estimate for π.

Step 3: Create a Function to Generate Random Points

Write a function that:

  • Generates a random x value between -1.0 and 1.0
  • Generates a random y value between -1.0 and 1.0
  • Stores these values in a Point struct
  • Returns the Point to the caller

Use C++’s random number facilities (not hard-coded values).

Step 4: Main Program Logic

In main():

  • Define the constant: constexpr double pi = 3.14159265;
  • Ask the user to enter the number of random points to generate
  • Use a for loop to generate that many points
  • Keep a count of how many points fall inside the circle (distanceFromOrigin() <= 1.0)

Step 5: Calculate π and Error

After generating all points:

  • Compute the Monte Carlo estimate of π
  • Calculate the error (variance) as the difference between your estimate and the defined constant
  • Output:
    • Total points generated
    • Estimated value of π
    • Error compared to the constant

Expected Behavior

As the number of points increases, your estimated value of π should become more accurate and the error should decrease.

This demonstrates the power—and limitations—of probabilistic methods.
