You will use the Monte Carlo method to estimate the value of π by randomly generating points and determining how many fall inside a circle. In this version, all points will be generated in the first quadrant from (0,0) to (1,1).

Step 1: Define the Point Struct

Use the Point struct shown below. This struct stores an x and y coordinate and also includes a member function that calculates the distance from the origin. 


struct Point
{
    double x;
    double y;

    double distanceFromOrigin() const
    {
        return std::sqrt(x * x + y * y);
    }
};

This demonstrates an important idea: although structs are often used to group data, they can also contain functions.

Step 2: Understanding the Monte Carlo Method

To estimate π, imagine a circle of radius 1 centered at the origin, inside a square that runs from -1 to +1 on both axes.

The area of the square will be 4 (since it's 2 units wide and 2 units tall), and the area of the circle will be π (since it's π * r^2 with r=1).

The ratio of the area of the circle to the area of the square is π / 4.

By randomly generating points in the square and counting how many fall inside the circle, we can estimate this ratio and thus compute an approximation for π.

For this assignment, we will only generate points in the first quadrant (from 0 to 1 on both axes), so we will be working with a quarter circle and a unit square.

  • Randomly generate points within the square
  • Check whether each point is inside the circle
  • The ratio of points inside the circle to total points approximates π / 4

From this ratio, you can compute an estimate for π.
