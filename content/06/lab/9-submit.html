Lab Submission Wrap-Up Checklist

Before submitting your work for all three student exercises, review the checklist below to ensure your lab is complete and ready for grading.

  • Verify required files
    Confirm that your lab folder contains only the files specified in each individual exercise (e.g., header files, implementation files, driver files, or the single required source file).
  • Check standard file headers
    Ensure every source and header file includes the required standard file header content exactly as specified in the course guidelines.
  • Review for magic numbers
    Avoid using unexplained numeric literals. Replace repeated or meaningful numbers with named constants (const or constexpr) when appropriate.
  • Compile and test your code
    Build and run each program to confirm it compiles without errors and produces the expected output.
  • Zip your lab folder
    Compress the entire lab folder (not individual files) into a single .zip file.
  • Submit to Canvas
    Upload the zipped lab folder to the appropriate Canvas assignment page before the deadline.

Taking a few minutes to complete this checklist can prevent avoidable point loss and ensure your work is evaluated fairly.
