Lab Introduction: Custom C++ Classes and Structs

In this lab, you will practice designing, writing, and using your own custom C++ classes and structs. These activities build directly on prior work with variables, functions, and objects, and introduce more realistic program structure and design.

Custom Classes

You will follow the provided instructions to define one or more custom classes. Each class will demonstrate core object-oriented concepts, including:

  • Private data members to protect internal state
  • Constructors to properly initialize objects
  • Getters to safely access attribute values
  • Setters to modify attribute values in a controlled way
  • A string representation method that outputs a readable description of the object (often called methods like print(), display(), or describe() in C++)

Through these exercises, you will see how classes bundle data and behavior together and how objects are created and used in a program.

Using Structs for Data Analysis

In addition to classes, this lab also includes an exercise using a struct. You will define a simple struct to group related data and functions, then use it as part of a data sampling experiment.

Specifically, you will apply a Monte Carlo simulation technique to estimate the value of π (pi). This exercise reinforces:

  • Using structs to represent simple data models
  • Generating random data samples
  • Applying mathematical logic using loops and functions
  • Comparing calculated results to known constants

Learning Outcome

By the end of this lab, you should be comfortable:

  • Creating and organizing custom C++ classes
  • Instantiating and working with objects
  • Writing clear interfaces using getters, setters, and output methods
  • Recognizing when a struct is an appropriate alternative to a class
  • Using code to perform basic data sampling and analysis

Be sure to follow all instructions carefully and include the required standard file header in each source file.
