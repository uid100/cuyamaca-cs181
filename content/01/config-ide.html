Week 1 Assignment: Configure Your C++ IDE

In this assignment, you will set up your C++ development environment so you can compile, run, and debug programs throughout the semester. Choose one of the recommended IDE options below.

Due Date: End of Week 1

Submission: Upload screenshots as instructed in each section.

Step 1: Choose Your IDE

Select one of the following:

  • Visual Studio Code (Windows, macOS, Linux)
  • Visual Studio Community (Windows only)
  • CLion (Windows/macOS/Linux — free with .edu email)
  • Replit (browser-based backup option)
Option A: Visual Studio Code Setup
  1. Install Visual Studio Code:
    https://code.visualstudio.com/
  2. Install the C/C++ Extension Pack by Microsoft.
  3. Install a compiler:
    • Windows: MinGW‑w64 or MSYS2
    • macOS: Install Xcode command‑line tools (xcode-select --install)
    • Linux: Install g++ via package manager
  4. Create a folder named cpp_projects on your computer.
  5. Create a new file hello.cpp and add a simple program.
  6. Build and run using the terminal.

Submission Required: Screenshot of VS Code showing hello.cpp and the output of your program.

Option B: Visual Studio Community Setup (Windows)
  1. Download and install Visual Studio Community:
    https://visualstudio.microsoft.com/vs/community/
  2. Select the C++ Desktop Development workload.
  3. Create a new Console App project.
  4. Add and run a simple hello world program.

Submission Required: Screenshot of Visual Studio with your program running.

Option C: CLion Setup
  1. Sign up for a free JetBrains student license:
    JetBrains Student License
  2. Install CLion for your OS.
  3. Verify that CMake and compiler toolchain are detected.
  4. Create a new Console Application and run a simple program.

Submission Required: Screenshot of CLion running your first program.

Option D: Replit (Backup Only)

Use this option only if a local install fails.

  1. Go to https://replit.com/
  2. Create a new C++ repl.
  3. Write and run hello world.

Submission Required: Screenshot of your browser showing your running output.
