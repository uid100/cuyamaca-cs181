Program 2: Print Formatted Address

Store the components of mailing addresses in separate variables and print the complete formatted addresses as shown.

Specifications:

File name:
address.cpp
Focus:
Use int, input, output, and string
Input:
Prompt the user for 1.) Name, 2.) street address, 3.) city, 4.) state, and 5.) zip code.
Store the sender's address components in separate variables.
Output:
The completed mailing label.
Example:
(input shown in square brackets)
    Enter name: [John Doe]
    Enter street address: [123 Main St]
    Enter city: [Anytown]
    Enter state: [CA]
    Enter zip code: [12345]
Example output:
    from:
    Cuyamaca College
    900 Rancho San Diego Pkwy
    El Cajon, CA 92019

    to:
    John Doe
    123 Main St
    Anytown, CA 12345
