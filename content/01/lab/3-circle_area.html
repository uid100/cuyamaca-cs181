Program 3: Area of a Circle

Write a C++ program that prompts the user to enter the radius of a circle, calculates the area, and displays the result.

Specifications:

File name:
circle_area.cpp
Focus:
Use int, input, arithmetic, and output
Input:
One double value entered by the user.
Store and use the value 3.14159 for pi.
Output:
The area of the circle.
Example:
        Enter radius: 5
        Area: 78.53975
  • Don't forget to test your program with different values to ensure it works correctly!
  • Be sure to use the value specified3.14159 for pi.
