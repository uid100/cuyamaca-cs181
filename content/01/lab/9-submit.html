Lab 01 — Submission Instructions

This lab consists of four C++ exercises. Each exercise must produce one .cpp file, using the filenames listed below.

Follow the steps on this page to prepare and submit your Lab01.zip file.

Required Files & Naming

Your Lab01 folder must contain these four files:

Lab01
├── address.cpp
├── circle_area.cpp
├── diamond_pattern.cpp
└── sum_integers.cpp
  • No subfolders inside Lab01 — all files must be at the top level.
  • Compiled executables may be present. They will be ignored and will not affect your grade.
  • Extra files (project files, temp files, etc.) will also be ignored.
Required File Header in Each .cpp

All your .cpp files must begin with the standard lab header.

Missing or incomplete headers may result in point deductions.

Creating Lab01.zip
Windows
  1. Right-click the Lab01 folder.
  2. Select Send to → Compressed (zipped) folder.
  3. Rename the resulting file to Lab01.zip.
macOS
  1. Right-click (or two-finger click) the Lab01 folder.
  2. Select Compress "Lab01".
  3. This creates Lab01.zip in the same directory.
Notes on Compilation

You are required to use a command-line C++ compiler such as g++.

Example compile command:

g++ sum_integers.cpp -o sum_integers

Executables may remain in the folder but will not be graded.

Final Submission

Upload Lab01.zip to the assignment dropbox.

Do NOT upload individual .cpp files — only the zip file.
