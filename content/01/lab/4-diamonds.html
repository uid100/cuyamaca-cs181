Program 1: Diamond Pattern

Write a C++ program to display a diamond pattern similar to exercise 2-12 in the textbook. as described below.

Specifications:

File name:
diamond_pattern.cpp
Focus:
Use console output
Input:
Ask the user to enter any printable character from the keyboard.
Output:
  • The diamond pattern should be seven (7) lines tall.
  • The diamond pattern should use the character entered by the user.
  • The diamond pattern should be centered and symmetrical.
  • Use only a single cout statement to display the diamond pattern.
Example:
  Enter any printable character: X
     X
    XXX
   XXXXX
  XXXXXXX
   XXXXX
    XXX
     X
