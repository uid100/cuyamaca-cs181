Getting Started With C++ Development

This guide walks you through setting up a simple C++ development environment and writing your first program using standard tools that work on any computer.

1. Install a C++ Compiler

Windows

  1. Install MinGW-w64 (Recommended):
    • Download from: https://www.mingw-w64.org
  2. Add mingw64/bin to your PATH.

macOS

  1. Open Terminal.
  2. Install developer tools: 
    xcode-select --install
  3. This provides clang++, a C++17-compatible compiler.

Linux

  1. Install g++: 
    sudo apt install g++

2. Install Visual Studio Code

VS Code is recommended because it is lightweight and works on every computer.

Download it here: https://code.visualstudio.com

Required VS Code Extensions

  • C/C++ (Microsoft)
  • Code Runner (optional but convenient)

3. Create Your First C++ Program

  1. Create a folder such as Lab1.
  2. Inside it, create a file named main.cpp.

Example Program

#include <iostream>
#include <string>
using namespace std;

int main() {
    string name;
    int age;

    cout << "Enter your name: ";
    cin >> name;

    cout << "Enter your age: ";
    cin >> age;

    cout << "Hello, " << name << "! You are " << age << " years old." << endl;

    return 0;
}

4. Compile and Run

Open a terminal inside your project folder and type:

g++ -std=c++17 -o program main.cpp

Run your program:

./program

5. Zip and Submit

  1. Zip your folder: 
    Lab1.zip
