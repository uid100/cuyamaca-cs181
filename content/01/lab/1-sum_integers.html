Program 1: Sum of Two Integers

Write a C++ program that prompts the user to enter two integers, calculates their sum, and displays the result.

Specifications:

File name:
sum_integers.cpp
Focus:
Use int, input, arithmetic, and output
Input:
Two integers entered by the user.
Output:
The sum of the two integers.
Example:
  Enter first integer: 5
  Enter second integer: 7
  Sum: 12
  • Don't forget to test your program with different integer values to ensure it works correctly!
  • Remember to include the required header files and use appropriate data types.
  • Include the required file header comment content as discussed.
